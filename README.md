# Python with Redis
votação simples em Python e conectado ao Redis

## Installing Redis on Ubuntu 20.04
`` sudo apt install redis-server ``

## Testing connection
`` sudo systemctl status redis-server ``

## Comands
### Delete
`` del KEY ``

### Delete all
`` flushall  ``
